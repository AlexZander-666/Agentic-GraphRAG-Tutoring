## 1. 金融场景 (Finance)

### 场景描述
从金融报告、投资分析、交易记录等文本中提取关键信息。

### 提取类别
- **机构**: 金融机构、公司、监管部门
- **人物**: 高管、分析师、投资人
- **金额**: 投资额、收益、估值等数字
- **时间**: 交易日期、报告期间
- **事件**: 融资、并购、上市等事件
- **指标**: 财务指标、增长率、市值等
- **产品**: 金融产品、投资标的

### 示例文本

```
【示例1：投资并购报告】

2024年12月15日，普华永道发布的《2024年中国金融科技投资报告》显示，今年前三季度，中国金融科技领域共发生投融资事件287起，总金额达到652亿元人民币，同比增长23.5%。

报告指出，蚂蚁集团以2850亿美元估值位居榜首，其次是京东科技（456亿美元）和微众银行（380亿美元）。值得注意的是，红杉资本中国基金在本季度参与了15笔交易，累计投资金额超过80亿元。

## 2. 医疗场景 (Medical)

### 场景描述
从病历、诊断报告、临床研究等医疗文本中提取结构化信息。

### 提取类别
- **患者**: 患者信息
- **症状**: 临床表现、主诉
- **诊断**: 疾病诊断结果
- **检查**: 检查项目、检查结果
- **药物**: 药品名称、用法用量
- **数值**: 检验数值、生命体征
- **时间**: 就诊时间、病程
- **医生**: 主治医生、专家

### 示例文本

```

```
【示例2：临床研究报告】

研究题目：EMPA-KIDNEY研究中国亚组分析——恩格列净对慢性肾脏病患者的肾脏保护作用

研究背景：
慢性肾脏病（CKD）是全球重大公共卫生问题，中国CKD患病率高达10.8%，约1.3亿人受累。SGLT2抑制剂恩格列净已被证实具有心血管和肾脏保护作用。

研究方法：
本研究为EMPA-KIDNEY研究（NCT03594110）的中国亚组事后分析。入组标准：eGFR 20-45 ml/min/1.73m²，或eGFR 45-90 ml/min/1.73m²伴尿白蛋白/肌酐比值（UACR）≥200mg/g。

中国亚组共纳入1268例患者，随机分配至恩格列净10mg组（n=634）或安慰剂组（n=634）。主要终点为肾脏疾病进展（定义为eGFR持续下降≥40%、进展至ESKD或肾脏原因死亡）或心血管死亡的复合终点。

主要研究者：
- 北京大学第一医院肾内科 赵明辉教授（首席研究者）
- 上海瑞金医院肾内科 陈楠教授
- 南京军区总医院肾内科 刘志红院士

研究结果：
中位随访2.8年后，恩格列净组主要终点事件发生率为15.8%（100/634），安慰剂组为23.2%（147/634），HR=0.65（95%CI: 0.50-0.84），P=0.001。

分层分析显示：
- 糖尿病亚组（n=486）：HR=0.62，绝对风险降低9.2%
- 非糖尿病亚组（n=782）：HR=0.68，绝对风险降低6.8%
- eGFR<30组（n=412）：HR=0.58，获益更为显著

安全性方面，恩格列净组酮症酸中毒发生率0.3%，生殖道感染发生率2.1%，均与既往研究一致。未观察到下肢截肢风险增加。

研究结论：
在中国CKD患者中，恩格列净10mg每日一次可显著降低肾脏疾病进展和心血管死亡风险，无论是否合并糖尿病均可获益。研究结果发表于《柳叶刀·糖尿病与内分泌学》2024年12月刊。
```

## 3. 销售场景 (Sales)

### 场景描述

- **客户**: 客户公司、联系人

- **时间**: 签约日期、交付时间
- **数量**: 采购数量、库存
- **需求**: 客户需求、痛点
- **竞争对手**: 竞品信息

## 4. 客服场景 (Customer Service)

### 场景描述
从客服对话记录、工单、投诉反馈等文本中提取关键信息，用于问题分析和服务改进。

### 提取类别
- **客户**: 客户姓名、会员等级、联系方式
- **订单**: 订单号、商品信息
- **问题**: 问题类型、问题描述
- **情绪**: 客户情绪状态
- **时间**: 问题发生时间、处理时限
- **解决方案**: 处理措施、补偿方案
- **责任方**: 问题归属部门
- **满意度**: 客户评价
